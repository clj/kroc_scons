Documentation of the API

The MySQL occam- API, or MoA for short, is built around the C API for MySQL. The API is developed for version 5.1 of the MySQL server, and is not guaranteed to work with older versions. MoA allows applications written in occam- to communicate with a MySQL server. It consists mainly of a pre-defined process definition, MySQL, with which you can create processes for communicating with a MySQL server. This is done by sending and receiving messages in the protocols MYSQL.CONTROL and MYSQL.RESULT. In addition, there is an initialising and a finalizing process, which will be of interest if you are to have more than one connection to MySQL open at a time. Note that, at the present, only the core functionality of the C API is available in MoA.
